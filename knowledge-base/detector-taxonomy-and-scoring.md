# AI渣男识别器分类与评分规则

## 总体原则

渣男指数不是“道德审判分”，而是“操控性关系风险分”。评分只回答：这段文本里出现了多少可证据化的操控话术、它们有多严重、是否形成重复模式。

## 评分总表

总分 100，8 类权重如下。0.2 版权重不是来自单篇论文的直接比例，而是基于已验证亲密伴侣暴力（IPV）/心理虐待量表与权威公共卫生定义做“证据映射 + 三轴归一化”：

- 伤害严重度：该类话术是否更容易侵蚀安全、自主性、自我价值、现实感或性/身体边界。
- 强制控制核心性：该类是否构成控制、隔离、威胁、羞耻、恐惧、依赖循环等核心机制。
- 测量证据强度：该类是否在 PMWI、CAS、CTS2、MMEA、coercive control 等量表/综述中有明确对应构念。

| 类别 | 上限 | 核心风险 |
|---|---:|---|
| PUA打压 | 12 | 削弱自尊，夺取评价权 |
| 煤气灯效应 | 16 | 扭曲现实感，制造自我怀疑 |
| 间歇性温柔 | 13 | 用奖惩循环维持依赖 |
| 暧昧分散 | 11 | 第三方稀释责任，模糊边界 |
| 道德绑架 | 15 | 用愧疚/羞耻/威胁迫使顺从 |
| 物化女性 | 14 | 将人降格为身体、性、服务或面子工具 |
| 拖延承诺 | 9 | 享受关系利益，推迟责任和定义 |
| 钓鱼撒网 | 10 | 维持多对象选项，低成本控场 |

详细来源和比例推导见 `weight-rationale.md`。

## 单类评分

每类按 0 到上限评分：

- 0：无证据。
- 1-3：弱信号，可能是沟通差或偶发失误。
- 4-6：中等信号，有明确原文证据，但频率或伤害有限。
- 7-9：强信号，多条证据或与行为模式一致。
- 10 分以上：严重信号，重复、跨场景、被指出后仍持续，或明显侵犯边界。
- 煤气灯效应可到 16：当现实否认、精神标签、孤立支持系统、证据反转同时出现时给高分。

## 分数区间

| 总分 | 风险等级 | 解释 |
|---:|---|---|
| 0-19 | 低 | 可能有误会、表达差异或普通冲突，暂不支持操控结论 |
| 20-39 | 轻中 | 有红旗，需要观察频率、行动和边界回应 |
| 40-59 | 中高 | 多类操控信号出现，建议设边界并降低情绪投入 |
| 60-79 | 高 | 形成较完整操控模式，建议认真评估止损和外部支持 |
| 80-100 | 极高 | 高度操控或安全风险，优先考虑安全、证据保存和求助 |

## 安全覆盖规则

如果出现以下任一情况，不管总分多少，都必须输出“安全优先”提示：

- 暴力威胁、跟踪、恐吓、限制人身自由。
- 性胁迫、反复突破性边界、酒精/药物相关性风险。
- 用自伤、自杀、伤害他人来逼迫用户顺从。
- 隔离用户朋友家人、监控手机、财务控制。
- 用户表达害怕、无法离开、被迫发生关系、被威胁曝光隐私。

安全提示不能命令用户“立刻分手”，但要建议联系可信任的人、本地专业机构或紧急服务，并保存证据。

## 8类判定细则

### 1. PUA打压

命中条件：

- 使用贬低、比较、讥讽、冷嘲削弱自尊。
- 把对方的正常需求说成幼稚、低价值、不配。
- 通过“打压后给一点认可”制造追逐。

高分证据：

- 外貌/年龄/性价值羞辱。
- “除了我没人要你”类隔离式评价。
- 多次在冲突中用贬低替代回应。

机制解释模板：

> 这类话术把评价权从你手里拿走，让你为了重新获得认可而解释、讨好或降低边界。

### 2. 煤气灯效应

命中条件：

- 否认已经发生的事实。
- 改写你的记忆和感受。
- 把你的质疑定义成心理问题。

高分证据：

- “你有病/你疯了/你精神不正常”。
- 明明有证据仍持续否认。
- 让你不敢相信自己的观察。

机制解释模板：

> 核心不是争论事实，而是让你怀疑自己是否有资格判断事实。

### 3. 间歇性温柔

命中条件：

- 伤害、冷淡、失联之后突然密集示好。
- 温柔只在你要离开或追责时出现。
- 承诺没有后续行动。

高分证据：

- 多轮“伤害 -> 道歉/甜言蜜语 -> 短暂变好 -> 复发”。
- 用“我离不开你”阻止你离开，但不解决问题。

机制解释模板：

> 不稳定的奖惩循环会让人等待下一次变好，从而降低离开的决心。

### 4. 暧昧分散

命中条件：

- 第三方关系定义和实际行为不一致。
- 用“只是朋友/同事/妹妹”最小化关系。
- 要求你大度，但拒绝透明边界。

高分证据：

- 隐瞒深夜聊天、暧昧称呼、约会或金钱往来。
- 把你的边界感说成小心眼、查岗、控制欲。

机制解释模板：

> 这类话术把焦点从“他是否越界”转成“你是否不够大度”，责任被稀释了。

### 5. 道德绑架

命中条件：

- 用愧疚、义务、牺牲叙事压迫你顺从。
- 把设边界说成不爱、不懂事、没良心。
- 用冷暴力、分手、自伤威胁迫使让步。

高分证据：

- “你要是真爱我就...”。
- “你逼我我就...”。
- 每次你表达需求，他都转为受害者。

机制解释模板：

> 它让你从判断事情对错，滑向证明自己是不是好人、是不是够爱。

### 6. 物化女性

命中条件：

- 把女性价值简化成身材、年龄、性、顺从、面子。
- 忽视意愿和边界，把陪伴/性/照顾当作应得。
- 使用性羞辱或年龄羞辱。

高分证据：

- “女人就该...”。
- “你这样带不出去/不值钱/没人要”。
- 对拒绝进行羞辱或报复。

机制解释模板：

> 物化会把你的主体性降成可使用资源，你的“不愿意”会被当成障碍而不是边界。

### 7. 拖延承诺

命中条件：

- 长期享受伴侣待遇，但拒绝明确关系或计划。
- 承诺总绑定不可控条件。
- 反复延期且没有行动。

高分证据：

- “等我忙完/有钱/稳定/我妈同意”反复出现。
- 每次追问未来都只给情绪安抚，不给可验证步骤。

机制解释模板：

> 未来承诺替代了当下行动，让关系停在对他成本最低、对你消耗最高的位置。

### 8. 钓鱼撒网

命中条件：

- 同时维持多个暧昧对象或潜在对象。
- 对每个人都给低成本情绪投喂。
- 对你忽冷忽热，保留外部选项。

高分证据：

- 多平台暧昧、深夜联系、暧昧备注。
- 被发现后用“我只是无聊/普通聊天/你管太多”解释。
- 你抽离时升温，稳定后降温。

机制解释模板：

> 它通过低成本维持多个可能性，让你持续竞争一个并不明确的位置。

## 加权修正规则

在基础 8 类分之外，应用以下修正，但总分封顶 100：

- 重复性：同类证据出现 3 次以上，+3。
- 跨类组合：煤气灯 + 道德绑架 + 间歇性温柔同时出现，+5。
- 边界回应：用户明确表达不舒服后对方继续，+5。
- 言行不一：承诺和行为冲突被证明，+3。
- 主动修复：对方承认问题、给出具体行动、持续兑现，可 -5 到 -15。
- 信息不完整：只有一两句、上下文缺失，最高风险等级不超过“轻中”，除非触发安全覆盖规则。

## 证据等级

每条证据标注可信度：

- A：原文直接命中，且上下文清楚。
- B：原文间接命中，需要结合前后文。
- C：来自用户概述，缺少原句。
- D：推测，不应计入高分，只能作为待确认点。

## 输出语气

应该坚定但不替用户做决定：

- 可以说：“这段话有明显的煤气灯特征。”
- 不要说：“他一定是 NPD/反社会人格。”
- 可以说：“如果这是反复模式，建议降低投入并设定可验证边界。”
- 不要说：“你必须马上分手。”
